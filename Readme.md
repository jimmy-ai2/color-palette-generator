# Color Palette Generator

A sleek and intuitive web application that helps designers and developers discover new color combinations.  
With one click, users can generate a harmonious palette of five random colors and copy HEX codes instantly.

---

## Features

- **Instant Generation**  
  Generates a fresh palette of five unique colors with a single click.

- **One-Click Copy**  
  Click a color block or icon to copy HEX values instantly.

- **Visual Feedback**  
  Copy icon changes to a checkmark to confirm success.

- **Responsive Layout**  
  Adapts seamlessly across mobile and desktop screens.

- **Modern UI**  
  Clean design with hover effects and smooth transitions.

---

## Tech Stack

- **HTML5** – Structure of the interface  
- **CSS3** – Styling using Grid, transitions, and modern design principles  
- **JavaScript (ES6+)** – Core functionality  
  - Random HEX color generation  
  - DOM manipulation  
  - Clipboard API for copy functionality  
- **Font Awesome** – Icons  
- **Google Fonts** – Poppins

## How to Use

1. **Generate Palette**  
   Click the *Generate Palette* button to create new colors.

2. **Explore Colors**  
   Hover over color blocks to focus on specific shades.

3. **Copy HEX Code**  
   - Click the color block  
   - Or click the copy icon  

4. **Confirm Action**  
   The icon turns into a checkmark when copied successfully.
