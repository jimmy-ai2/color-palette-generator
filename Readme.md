# Color Palette Generator

## About the Project

The **Color Palette Generator** is built to solve the "blank canvas" problem for creatives. By leveraging Vanilla JavaScript and modern CSS techniques, it offers a lightweight, fast, and visually pleasing way to explore color theory. Users can generate fresh palettes instantly and grab the Hex codes for their projects without manually typing them out.

## Features

- Instant Generation of colors with one click
- Click-to-Copy function

## Tech Stack

- HTML
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## Possible Improvements

- **Color Locking:** Add a "lock" icon to each color so users can keep favorites while regenerating the rest of the palette.
- **Export Options:** Allow users to download the palette as a CSS file, SCSS variables, or a JSON object.
- **Theme Integration:** Add the ability to generate palettes based on specific moods (e.g., "Neon," "Pastel," or "Earth Tones").
- **Local Storage:** Save a history of generated palettes so users can revisit previous combinations.

## How to Use

1.  **Clone the repository:**
    ```
    git clone https://github.com/jimmy-ai2/color-palette-generator.git
    ```
2.  **Open the project:**
    ```
    Simply open `index.html` in your preferred web browser.
    ```
3.  **Generate:**
    ```
    Click **"Generate Palette"** or tap any color to copy its Hex code!
    ```
